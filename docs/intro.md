# 手册说明

MingYSub 字幕组手册旨在介绍字幕制作的完整流程，适用于没有字幕制作经验的初学者。内容涵盖翻译、校对、时轴、后期、特效、压制、发布等主要环节，从基础知识入手，逐步展开。

指南中出现的“参考规范”，默认都是指 MingYSub 字幕组的规范，不代表其他字幕组。
